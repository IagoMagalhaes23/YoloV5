# YoloV5

Que tal treinar o modelo YoloV5 com o dataset COCO128? 🎉🎉🎉

Objetivos:

✅ Conhecimento básico em detecção de objetos, YOLO e COCO. Em seguida, configurar seu notebook e começar a explorar seus dados.

✅ Revisar as técnicas de treinamento de rede neural e as métricas de avaliação para detecção de objetos enquanto treinava e validava o modelo.

✅ Usando o moviepy, você selecionamos imagens não vistas pelo modelo durante o treinamento e as concatenou para produzir um vídeo de inferência. 

Ao longo do desafio, você também desenvolveu habilidades universais de ciência de dados, incluindo pensamento estruturado, experimentação, solução de problemas, treinamento de modelos complexos e trabalho com dados em grande escala, para citar alguns.
